# Web3

Lumi is an exemplary use case for Web3, operating as a Web3 assistant powered by Web3Go's Intelligent ML models and the latest GPT model.&#x20;

It has been trained using web project documentation and various other sources of information. Lumi's primary objective is to provide users with a comprehensive understanding of project information, encompassing the latest updates from the Internet and on-chain data.

By leveraging Lumi, users can stay informed about the latest developments in the Web3 world, enabling them to make more informed decisions regarding their investments and projects.

Currently, Lumi has been onboarded for various Web3 projects, including BNB Smart Chain, Lifefrom, Yuliverse, SecondLive, and more.
